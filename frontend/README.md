# Frontend skeleton (Vue 3) - Dashboard + Treasury

Đây là skeleton để bạn bù đúng yêu cầu bản thường:
- Dashboard: pinned news + treasury summary + top 5 ranking + cảnh báo quỹ âm.
- Treasury: categories + transactions.

Bạn cần map lại theo cấu trúc project Vue hiện tại (router/store/api baseUrl).
